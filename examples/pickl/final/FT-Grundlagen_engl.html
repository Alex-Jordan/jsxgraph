2 Basics

  1. What is the FT/ IFT?

    The Fourier transform (FT) is an integral transform, which attaches a suitable original function f(t) to its image function or Fourier transformed F(ν).
    The original function f(t) is piecewise continuous and absolute integrable. Typical its domain is time.
    The image function F(ν) is a function of frequency ν.

    The inverse Fourier transform (IFT) is the inverse function of the Fourier transform.
    It transforms an image function F(ν) into its original function f(t). By using the IFT a function of time can be generated from a frequency spectrum.

    In practice audio signals are scanned with a "scan comb" (delta pulse sequence) in equidistant time intervals tn = nΔt with n = 0, 1, 2, ..., m and the scan time Δt > 0, so that a discrete signal is produced. So the function exists as measuring points and is described as the sequence fn.
    Here you have to notice, that the scan times Δt need to be chosen small enough. Then the scanned sequence fn reflects the signal adequate, so that the signal can be reproduced out of the measuring points.

    In the modern computer science a fast, very effective algorithm, the fast Fourier transform (FFT), is adopted in order to calculate discrete Fourier transforms with a high m. The number of realized computations using the FFT is proportional to N·log2(N). This is much less than using the DFT with its ∝N² operations.
    The inverse function of the FFT is the inverse fast Fourier transfrom (IFFT).

  4. Specialties in application of the FFT in signal processing

    In signal processing audio signals are scanned with a "scan comb" (Delta pulse seqeunce) to produce a discrete signal. So generally 44100 measured values are scanned per second. When scanning there is always a refelection to 0.5 · sample rate,
    In other words 0.5 · 44100 Hz = 22050 Hz, what conforms to the maximal hearable frequency domain of the human ear.

    The reason for the reflection is made clear in the following image:

    The upper graphic shows two different waves, one with 0.3 · sample rate and one with 0.7 · sample rate. The red lines describe those measured values, that are saved during the scanning. You can see that both waves result in the same scanned vaules. Hence, the scanned signals as well as their frequency spectra after FFT are always symmteric to 0.5 · sample rate. For this reason only the first half of the frequency spectrum of the transformed function is relevant.
