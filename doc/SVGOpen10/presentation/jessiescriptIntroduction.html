• Idea:
    Easy to learn scripting language to use in the classroom with students and pupils to display math with JSXGraph - instead of requiring programming skills in JavaScript.
  • Alternative: JessieScript
    JessieScript was invented at the university of Bayreuth and is a syntax similar to what is taught in schools and can be parsed by JSXGraph.
  • Examples:
    P(1,1); Q(-2,2); g=[PQ]; k(P,2); M=1/2(P,Q); ||(g,M); f:x^2+2*x-3;
  • Additional positive aspect:
    No neccessity to let a user write JavaScript code to produce constructions, i.e. it is possible to let any user provide constructions e.g. in a wiki in a secure way because everything gets parsed by JSXGraph.
