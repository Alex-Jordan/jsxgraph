JSXGraph can be used on every device which runs at least one of the supported browsers, i.e.

  • PC (Windows, Linux, Mac)
    using SVG where it is supported and VML for the Internet Explorer
  • "Touchpads" like the Apple iPad
    enabling touch events for dragging and even some multitouch possibilities
  • Mobile phones, iPod
    because no (Java or Flash) plugin is required
  • Android devices
    using a CanvasRenderer (available since version 0.82rc8, still under development)
